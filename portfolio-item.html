iCamp App

Fictional company Web page listing selected campgrounds around Ireland. It displays location, image and price. Also, a small description of the selected campground.

Web page requires user to sign up and log in in order to leave a comment and upload a campground to the web site, as well, to edit and delete their own campgrounds and comments.

Uses Express, MongoDB, RESTful Routing, User Authentication, and Bootstrap

Final Project of "The Web Developer Bootcamp" - Udemy
